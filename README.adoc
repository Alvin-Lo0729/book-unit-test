== 單元測試

=== 單元測試不是測試

=== 單元測試的起手式:人生第一個單元測試

==== 單元測試

. 單元測是要解的是一個邏輯單元功能是否正確，這短短的一句話，就有兩個重點
(1) 什麼叫一個單元?
(2) 什麼叫正確?

. 單元測是真正做的事情，就是檢測一段程式碼「跑起來的表現」，跟「在我腦中想像的表現」是否吻合
. 至於什麼是一個單元，作者認為，只要你認為他「足以表達一個商業邏輯，而且只表達這個商業邏輯」，就可以稱為單元
